Enclosed is a Windows Console Application created using	Visual Studio 2017 Community Edition, Version 15.8.7
The Target Framework is .NET Core 2.1
Source code can be found in the /src folder
A self-contained x64 version of the application can be found in the /app folder
The application can be executed via one of the following:
	Navigate to the app folder in Windows Explorer and open the MagazineStore.exe file directly
	Navigate to the app folder in a Command Prompt and type 'magazinestore' into the Console followed by the Enter key
